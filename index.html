Structure

7Host Template is well commented to easily understand, everypage has
  1. < head > area which contains the meta tags and css files
  2. < body > area which contains the site and divided into
  3. < header > with id "header" and includes menu, logo area
  4. < footer > with id "footer" includes all footer content .

HTML Structure

  1. To set the boxed page add in Body class body-boxed
  2. To set the boxed page add in Body class body-boxed-2
  3. To set the fixed nav page add in #header class fixedHeader
